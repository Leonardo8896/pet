# Descrição

O projeto Print Experience Technology (PET) desenvolve uma máquina capaz de reciclar garrafas PET por meio de extrusão controlada, transformando-as em filamento de alta precisão para impressoras 3D. Integrando eletrônica embarcada, controle térmico via PID, automação com microcontrolador STM32 e um sistema mecânico de tração aprimorado com encoder magnético, o protótipo garante estabilidade dimensional, eficiência energética e operação contínua. Além de apresentar forte viés ambiental — ao reduzir resíduos plásticos e promover a economia circular — o sistema foi projetado para ser acessível, replicável e educacional, possibilitando seu uso em escolas técnicas, ambientes maker e projetos de baixo custo voltados à inovação sustentável.
